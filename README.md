# 📘 VEDIC MATHEMATICS TRICKS (MULTIPLICATION)

---

## 🔷 1. Base Method (Nikhilam Method)

### 📌 Concept

Used when numbers are **close to a base** (10, 100, 1000, etc.)

---

### 🧠 Steps

1. Choose base (nearest power of 10)
2. Find **deficiency or excess**
3. Crosswise add/subtract → Left part
4. Multiply vertically → Right part
5. Apply **placement rule**

---

### 📌 Example

```
98 × 97

Base = 100
98 | -02
97 | -03

Step 1: 98 - 03 = 95 OR 97 - 02 = 95
Step 2: (-2 × -3) = 06

Answer = 9506
```

---

### ⚠️ Placement Rule

If base = 100 → 2 digits on right
If base = 1000 → 3 digits on right

---

### 📌 Example (Carry Case)

```
7 × 6

Base = 10
7 → -3
6 → -4

Step 1: 7 - 4 = 3
Step 2: (-3 × -4) = 12

→ write 2, carry 1

Answer = 42
```

## 🔷 2. Numbers Above Base

### 📌 Concept

When numbers are **greater than base**

---

### Example

```
12 × 13

Base = 10
12 → +2
13 → +3

Step 1: 12 + 3 = 15
Step 2: 2 × 3 = 6

Answer = 156
```

---

## 🔷 3. Squaring Numbers (Near Base)

### 📌 Concept

Used for fast squares like 101², 103²

---

### Formula

```
(x + a)² = (x + a + a) | (a²)
```

---

### Example

```
103²

Excess = 3

Step 1: 103 + 3 = 106
Step 2: 3² = 09

Answer = 10609
```

---

## 🔷 4. Above & Below Base

### 📌 Concept

One number above, one below base

---

### Example

```
15 × 8

Base = 10
15 → +5
8 → -2

Step 1: 15 - 2 = 13
Step 2: 5 × (-2) = -10

Step 3: 13 × 10 = 130 - 10

Answer = 120
```

### 📌 Concept

Used for subtraction from powers of 10

---

→ 9-1 = 8
→ 9-2 = 7
→ 10-3 = 7

## 🔷 6. Working Base (Multiples like 50, 200)

### 📌 Concept

Use when numbers are near **non-power bases**

---

### Example

```
44 × 48

Base = 50

44 → -6
48 → -2

Step 1: 44 - 2 = 42
Step 2: (-6 × -2) = 12

Adjust: 42 × 5 = 210
Add carry → 2112

Answer = 2112
```

### 📌 Concept

When numbers have **different bases**

---

Divide by 10

Answer = 91233

## 🔷 8. Vertically & Crosswise Method

### 📌 Concept

Universal method → works for ALL numbers

---

### 🔹 2-digit × 2-digit

#### Example

```
12 × 43

Step 1: 2×3 = 6
Step 2: (1×3 + 2×4) = 11 → write 1, carry 1
Step 3: 1×4 + 1 = 5

Answer = 516
```

---

### 🔹 2-digit × 1-digit

#### Example

```
34 × 8

Make → 34 × 08

Step 1: 8×4 = 32 → write 2, carry 3
Step 2: (8×3 + 0×4) + 3 = 27
Step 3: 0×3 + carry = 2

Answer = 272
```

---

### 🔹 3-digit × 3-digit

#### Example

```
123 × 456

Step 1: 3×6 = 18 → write 8, carry 1
Step 2: (3×5 + 2×6) = 27 +1 = 28 → write 8, carry 2
Step 3: (3×4 + 2×5 + 1×6) = 28 +2 = 30 → write 0, carry 3
Step 4: (2×4 + 1×5) = 13 +3 = 16 → write 6, carry 1
Step 5: 1×4 +1 = 5

Answer = 56088
```

---

### 🔹 3-digit × 2-digit

#### Example

```
321 × 42 → make 042

Apply same method

Answer = 13482
```
